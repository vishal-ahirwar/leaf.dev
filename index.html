Everything You Need

Leaf provides a robust set of commands to completely handle the C++ lifecycle—all zero-config.

Project Setup & Environment

leaf create

Generates a new, fully structured Leaf project in a new directory, ready for code.

leaf init

Initializes a new Leaf project structure within an existing directory.

leaf setup

Setup Clang Toolchain for C/C++ Development including cmake, ninja, conan, and clang.

leaf doctor

Checks your system to ensure all required tools (Clang, CMake, Conan) are properly installed.

Build & Code Quality

leaf build

Compiles the entire project and perfectly links all its dependencies.

leaf run

Compiles and executes the main application target of your project in one step.

leaf compile

Builds all targets in the project without running them.

leaf clean

Deletes all build artifacts and reconfigures the project from scratch.

leaf format

Automatically formats all C++ source code files according to a consistent style.

leaf docs

Automatically generates extensive documentation for your project using doxygen.

leaf tests

Runs all test suites project-wide using ctest.

Dependency Management

leaf install

Fetches and installs all the dependencies listed in the underlying conanfile.py.

leaf addpkg

Add an external library to your project from the central registry.

leaf remove

Safely remove a package dependency from your project.

leaf addapp

Add a new executable sub-project to your workspace.

leaf addlib

Add a shared library to your workspace.

leaf search

Search for packages in the Conan package registry.

leaf info

Display detailed information about a specific package.

leaf tree

Display a visual dependency tree of the current active project.

Distribution & Hosting

leaf publish

Creates a release-ready Conan package from your completed project.

leaf upload

Uploads a packaged library to a specified remote Conan repository.

leaf server

Manages a lightweight Leaf package server for natively hosting private Conan packages.

leaf update

Check for Leaf updates and seamlessly download the latest version from GitHub.

Install Leaf

Setup Leaf in seconds directly from your terminal. Ensure you have C++20, CMake, and Conan ready.

# Install via PowerShell
powershell -NoProfile -ExecutionPolicy Bypass -Command "Invoke-WebRequest -Uri https://raw.githubusercontent.com/vishal-ahirwar/leaf/master/install.bat -OutFile install.bat; .\install.bat"
# Install via Curl
curl -fsSL https://raw.githubusercontent.com/vishal-ahirwar/leaf/master/install.sh | bash
# Clone and build using CMake
git clone https://github.com/vishal-ahirwar/leaf.git
cd leaf
cmake -B build
cmake --build build

After installation, verify with:

leaf version
